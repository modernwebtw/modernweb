Modern Web 2018

  • 1

    一想到網站技術大會，就會想到 ModernWeb

    連續 3 年大受好評，2018 年 為期 2 天的會期，500 人與會，超過 20 場專業演講，精彩的議程內容，更多大師登場，前端、後端、架構，三大主題全面展開，來自中國的 Web 技術分享。還有滿滿的場邊活動，將再次打造全新的「心」感受。

  • 2

    首度登台！國際大師坐了一天飛機也要來！

    你一定都有看過他寫的書，同時也是 Web 開發者的 Jeremy Keith，首度從英國遠渡重洋來台，帶來最新的觀點，最精彩的演說，錯過這一次，可能要再等 4 年；來自於美國 Slack 的 Felix Rieseberg ，喜歡吃糖果的大男孩，投入在 Open Source 的懷抱中，將在這首次登上的舞台，帶來豐富的議程演說。

  • 3

    電腦不會挑選議程，但是我們會！

    大會的的議程 200% 都是由專業團隊嚴選，今年 ModernWeb18 講者投稿熱烈，在眾多投稿中仔細審核內容、反覆討論而定案。為滿足更多 Web 人的需求，2018 年，特別規劃，從前端、後端及架構三大主題，滿滿的 2 天，滿滿的收獲。

  • 4

    前端巧手、後端好手、全端高手，全台好手齊聚！

    參加研討會，除了專心聽議程吸收經驗之外，最重要的當然是把握現場與其它好手交流、還有與專家面對面討教的機會。在這個場域中，將聚集了全台的各路好手，不論是技術交流、認識朋友、吸收知識或是獲取經驗，都是非常難能可得寶貴的機會。

  • 5

    好吃、好玩、又好拿！滿足你的腦子與肚子

    研究指出，大腦就算「甚麼也不想」，每分鐘也會消耗0.1卡，若是在思考較複雜的事情時，每分鐘甚至會燃燒1~1.5卡，為了維持大腦正常的運作，將供應各種補充能量的「補給品」，累了嗎？補充能量再上；當然還有不能錯過的 ModernWeb 18 大會紀念服等你來收集！

Workshop
前端的速度與激情

用 Angular 與 Firebase 快速建構網站雛形
實戰領航員
楊捷凱 (Kevin)
領航簡介

目前擔任 Angular Taiwan 主要活動籌備者、Angular 線上讀書會主持人、Angular 論壇站長，熱衷研究於Angular 實務上可能的應用變化，不定期舉辦Angular進階議題佛系工作坊。
實戰時間 2018/07/22(日) 09:00 - 17:00
實戰大綱

Angular 是一套純前端的網頁開發框架，搭配 Firebase 更能讓我們在短的時間內建構出一個完成的網站雛形，且不需要任何後端技術支援，一切都透過 Firebase 提供的 API 與雲端資料庫搞定所有需求、快速實驗想法。



本次上手實作將帶領大家快速上手 Angular 與 Firebase 實作，你不需要有任何 Angular 或 TypeScript 的相關知識，讓我們用一天的時間，帶你瞭寫 Angular 與 Firebase 的神秘面紗。
實戰收獲
  • 了解 Angular 如何開發前端網站
  • 了解 Firebase 的架構與應用方式
  • 了解前端人如何運用工具快速建構模型
